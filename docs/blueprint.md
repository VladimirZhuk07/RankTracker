# **App Name**: CS2 Rank Tracker

## Core Features:

- User Ranking Display: Display a list of CS2 users with their ratings, sorted in descending order.
- Avatar Display: Show each user's avatar alongside their name and rating on the main board.
- Admin Authentication: Secure admin panel accessible only with valid username and password.
- User Creation: Allow the admin to create new users via the admin panel.
- Avatar Update: Enable the admin to update user avatars through the admin panel.
- CSV Rating Upload: Functionality for the admin to upload a CSV file to update user ratings.

## Style Guidelines:

- Primary color: A desaturated light grey (#D3D3D3) evoking the industrial and gritty feel of CS2.
- Background color: A dark grey (#333333) for contrast and readability.
- Accent color: A vibrant orange (#FFA500), drawing from the game's explosive effects and providing highlights for interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headings, and 'Inter' (sans-serif) for body text, for a modern and readable design.
- Responsive design with a fluid grid layout for optimal viewing on different screen sizes.
- Use clean, simple icons for user actions and navigation.
- Subtle transitions and hover effects to enhance user interaction.
